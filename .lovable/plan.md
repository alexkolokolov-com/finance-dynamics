## Цель
Ужать hero страницы `/event` на мобильном и планшете так, чтобы кнопка «Зарегистрироваться» умещалась в первый экран. Десктоп оставить как есть.

## Структура hero на mobile/tablet (< lg)

```text
┌───────────────────────────┐
│  [фото фоном, aspect 4/5] │
│  ▸ пилюли (сверху)        │
│  ▸ градиент вниз          │
│  ▸ подзаголовок (снизу):  │
│    Бизнес-завтрак         │
│    с Василием             │
│    Мещеряковым            │
└───────────────────────────┘
   H1 (заголовок встречи)
   [ Зарегистрироваться → ]
```

Всё в один экран (~100svh с учётом header'а).

## Что меняется в `src/pages/Event.tsx`

1. Убираю текущий отдельный ряд пилюль над сеткой (mobile-only) — они переезжают на фото.
2. На `< lg` рендерю новый компакт-блок:
   - Контейнер: `relative w-full aspect-[4/5] max-h-[62svh] overflow-hidden border hard-shadow`.
   - `<img>` фоном, `object-cover`.
   - Верхний слой: пилюли `Calendar` и `MapPin` — `absolute top-3 left-3 right-3 flex flex-wrap gap-2`, с полупрозрачным фоном `bg-background/85 backdrop-blur` чтобы читались поверх фото.
   - Нижний градиент: `absolute inset-x-0 bottom-0 h-1/2 bg-gradient-to-t from-background via-background/70 to-transparent`.
   - Подзаголовок поверх градиента: `absolute bottom-4 left-4 right-4 font-serif-display text-xl leading-tight` с явными переносами строк:
     `Бизнес-завтрак`<br/>`с Василием`<br/>`Мещеряковым`.
     Разбивка задаётся `<br />`, чтобы совпадало на любых ширинах mobile/tablet.
3. Под фото: H1 (уменьшенный clamp, `text-[clamp(1.75rem,6vw,3rem)]`, `mt-6`) и кнопка «Зарегистрироваться» (`mt-5`). Декоративная рамка/квадрат/шильдик «River Loft · Москва» на mobile скрываются.
4. На `lg` — оставляю текущую двухколоночную раскладку без изменений (`hidden lg:grid` для десктопной сетки, `lg:hidden` для нового мобильного блока).
5. Уменьшаю верхний паддинг секции на mobile: `pt-24 md:pt-32 lg:pt-40`, `pb-10 md:pb-16 lg:pb-24`.

## Технические детали
- Пилюли поверх фото используют существующий класс `badge-tag`, но переопределяю фон на `bg-background/85 backdrop-blur-sm border-foreground/15` через локальные утилиты, чтобы не трогать глобальный компонент.
- Неразрывные пробелы в подзаголовке проставляю через `nbsp()`, но переносы строк задаю руками (`<br />`), т.к. точное разбиение по строкам критично.
- Фото — тот же `heroPhoto.url` из `@/assets/vasya-brunch.jpg.asset.json`.
- Изменения только в `src/pages/Event.tsx`, ничего в бизнес-логике и других страницах не трогаю.
